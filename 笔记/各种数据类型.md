## 无符号和有符号的区别

首先无符号为正数， 取值范围都大于0

有符号数包括负数，无符号数只有整数

取值范围：

整型范围¶

    Int8 - [-128 : 127]
    Int16 - [-32768 : 32767]
    Int32 - [-2147483648 : 2147483647]
    Int64 - [-9223372036854775808 : 9223372036854775807]

无符号整型范围¶

    UInt8 - [0 : 255]
    UInt16 - [0 : 65535]
    UInt32 - [0 : 4294967295]
    UInt64 - [0 : 18446744073709551615]

参阅文档

https://clickhouse.yandex/docs/zh/data_types/int_uint/

使用建议：

mysql 中默认是有符号， 当确定是整数的时候，使用无符号即可
